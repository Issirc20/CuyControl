# Manual de Usuario

## Introducción

CuyControl es una aplicación de gestión integral para granjas de cuyes. Permite administrar cuyes, alimentos, ventas y generar reportes.

## Acceso a la Aplicación

1. Abrir navegador y acceder a: http://localhost:5000
2. Introducir credenciales (usuario y contraseña)
3. Seleccionar rol de acceso (si aplica)

## Módulos Principales

### Dashboard
- Vista general de la granja
- Estadísticas principales
- Alertas de eventos importantes

### Gestión de Cuyes
- Registrar nuevos cuyes
- Actualizar información
- Consultar historial de pesos
- Visualizar eventos sanitarios

### Alimentación
- Registrar alimentaciones por jaula
- Consultar histórico
- Generar reportes de consumo

### Inventario
- Controlar stock de alimentos
- Registrar movimientos
- Alertas de stock bajo

### Ventas
- Registrar ventas
- Consultar historial
- Generar reportes de ingresos

### Reportes
- Descargar en PDF o Excel
- Filtrar por fecha
- Análisis de mortalidad y reproducción

## Roles de Usuario

- **Administrador**: Acceso total, gestión de usuarios y configuración.
- **Operador**: CRUD de entidades, excepto configuración.
- **Veterinario**: Solo lectura y reportes de salud.

## Troubleshooting

- Si no puedes acceder: Verificar credenciales.
- Si el dashboard no carga: Limpiar caché del navegador.
- Errores de base de datos: Contactar administrador.
