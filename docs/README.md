- CHANGELOG.md: Historial de versiones.
- ROADMAP.md: Hoja de ruta del proyecto.
- ARQUITECTURA.md: Descripción arquitectónica.
- CASOS_DE_USO.md: Casos de uso principales.
- REQUERIMIENTOS.md: Requisitos funcionales y no funcionales.
- MANUAL_TECNICO.md: Guía para desarrolladores e integraciones.
- MANUAL_USUARIO.md: Guía de usuario final.
- API_DOCS.md: Documentación de endpoints públicos.
- DATABASE_SCHEMA.md: Esquema de base de datos.
- SEGURIDAD.md: Consideraciones de seguridad.
- TESTING.md: Indicaciones para ejecutar pruebas.

Mantener estos documentos actualizados es responsabilidad del equipo de desarrollo.
