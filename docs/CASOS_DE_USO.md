# Casos de Uso

1. Autenticación y autorización: Usuarios con roles (Administrador, Operador, Veterinario).
2. Gestión de cuyes: CRUD de registros de cuyes, pesos y control sanitario.
3. Alimentación: Registrar y consultar alimentaciones por jaula.
4. Inventario de alimentos: Controlar stock y movimiento de inventario.
5. Ventas: Registrar ventas y generar reportes.
6. Reportes: Dashboard y exportación de datos.
